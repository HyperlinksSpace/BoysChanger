Built for live voice

One engine, many characters — toggled with a single ON/OFF control.

  • Character stack Race, gender, age, timbre, amplifier, and volume.
  • Layered effects Echo, wah-wah, distortion, reverb, chorus, robot, flanger, bitcrush — all at once.
  • 11s prehear Replay the last eleven seconds of processed voice before you go live.
  • System-wide input Route through a virtual cable and set it as the OS microphone.

Quick setup

Windows

  1. Install VB-Cable and reboot.
  2. Install BoysChanger from the latest release.
  3. Set Output to CABLE Input.
  4. Click Apply as system input (or set default mic to CABLE Output).
  5. Flip the changer ON.

macOS

  1. Install BlackHole 2ch.
  2. Install BoysChanger from the latest release.
  3. Set Output to BlackHole 2ch.
  4. Click Apply as system input (or set input to BlackHole in Sound settings).
  5. Flip the changer ON.
